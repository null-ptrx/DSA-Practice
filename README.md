# Data Structures and Algorithms (DSA) in C++

Welcome to my DSA learning repository! This repo contains my journey of learning Data Structures and Algorithms using C++, organized topic-wise for easy navigation.

## 📚 Topics Covered

### 1. **Time and Space Complexity**
- Understanding Big O, Big Omega, and Big Theta notations
- Analyzing time complexity of algorithms
- Space complexity analysis
- Best, Average, and Worst case scenarios
- Practical examples and comparisons

### 2. **Pattern Questions**

Covered 22 pattern questions to improve loop logic

### 3. **C++ STL (Standard Template Library)**
- **Containers:**
  - Vector
  - Stack
  - Queue
  - Deque
  - List
  - Set / Multiset
  - Map / Multimap
  - Unordered Set / Unordered Map
  - Priority Queue
  
- **Algorithms:**
  - Sorting (sort, stable_sort)
  - Searching (binary_search, lower_bound, upper_bound)
  - Manipulation (reverse, rotate, swap)
  - Numeric operations (accumulate, min, max)

- **Iterators:**
  - Forward, Backward, and Random access iterators
  - Iterator operations and usage

### 4. **Basic Mathematics for DSA**
- **Number Theory:**
  - Counting digits
  - Reversing numbers
  - Palindrome numbers
  - Armstrong numbers
  - Prime numbers
  - Divisors of a number
  - GCD (Greatest Common Divisor)
  - LCM (Least Common Multiple)
  
- **Mathematical Operations:**
  - Modular arithmetic
  - Power functions
  - Factorial calculations
  - Euclidean algorithm

## 🚀 How to Use This Repository

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/DSA-CPP.git
   ```

2. Navigate to the specific topic folder:
   ```bash
   cd DSA-CPP/Basic-Maths
   ```

3. Compile and run any C++ file:
   ```bash
   g++ filename.cpp -o output
   ./output
   ```

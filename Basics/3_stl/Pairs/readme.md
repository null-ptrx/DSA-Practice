# DSA Series: Standard Template Library (STL) - Pairs

Welcome to my DSA learning repository! This section focuses on **`std::pair`**, a simple yet powerful container in C++ used to combine two values of different or same types into a single unit.

## Overview
Pairs are extensively used in competitive programming and algorithm design, particularly when:
- Storing coordinates (x, y).
- Managing edges and weights in **Graph Theory**.
- Returning two values from a single function.
- Storing key-value pairs before moving to `std::map`.

## Code Snippet
The provided C++ file demonstrates:
1. **Initialization**: Using constructors, `make_pair()`, and curly brace initialization.
2. **Accessing Data**: Utilizing the `.first` and `.second` members.
3. **Comparison**: How pairs are compared lexicographically.
4. **Arrays of Pairs**: Implementing pairs within an array structure.

## How to Run
To compile and run the code, use a C++ compiler like `g++`:

```bash
g++ pairs_learning.cpp -o pairs
./pairs
```
